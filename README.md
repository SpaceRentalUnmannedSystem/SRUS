# SRUS(SpaceRentalUnmannedSystem)
---
##프로젝트 구성
###프로젝트 명 : IoT를 활용한 공간 대여 무인 시스템
###구현 목표 : IoT를 이용한 인증을 통해 무인 대여함을 열고 닫을 수 있는 무인 대여함 시스템

---

###분석(2종)
1. 요구사항 정의서
2.요구사항 분석서

###설계(13종)
1. 시스템 구성도
2. 엔티티 정의서
3. 사전 정의서
4. 개념 ERD
5. 논리 ERD
6. 테이블 정의서
7. 물리 ERD
8. 유즈케이스 모델 정의서
9. 인터페이스 정의서
10. 프로세스 정의서
11. 프로세스 설계서
12. 화면 정의서
